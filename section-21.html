Seção 21 Revisão sobre números complexos.

Definição 21.1.

Vamos representar por \(i\) a unidade imaginária  1  satisfazendo \(i^2 = -1\text{.}\) Definimos então os números complexos como os números da forma

\begin{equation*} a + b i, \,\,\,\,\,\, a,b \in \mathbb{R}, \end{equation*}

onde \(a\) é dita a parte real e \(b\) a parte imaginária. Denotamos o conjunto de todos os números complexos por \(\mathbb{C}\text{.}\)

Também chamada de número imaginário.

A operação de soma (analogamente para a subtração) de dois números complexos \(z=a+bi\) e \(w=c+di\) é definida adicionando parte real com parte real e parte imaginária com parte imaginária, como se estivéssemos somando dois polinômios do primeiro grau na variável \(i\text{:}\)

  • \(\displaystyle z+w = (a+bi)+(c+di) = (a+c)+(b+d)i \in \mathbb{C},\)

  • \(\displaystyle z-w = (a+bi)-(c+di) = (a-c)+(b-d)i \in \mathbb{C}.\)

A multiplicação de números complexos é definida por \(z\cdot w = (a+bi)\cdot(c+di)=(ac-bd) + (ad+bc)i \in \mathbb{C}\text{,}\) como se estivéssemos multiplicando dois polinômios do primeiro grau na variável \(i\text{:}\)

\begin{equation*} z\cdot w = (a+bi)\cdot(c+di) = ac+adi+bci+bdi^2=(ac-bd) + (ad+bc)i. \end{equation*}

O conjugado de um número complexo \(z=a+bi\) é o número complexo \(\bar{z}=a-bi\text{.}\)

A divisão de números complexos é definida por

\begin{equation*} \frac{a+bi}{c+di} = \frac{ac+bd}{c^2+d^2} + \frac{bc-ad}{c^2+d^2} i \in \mathbb{C}, \end{equation*}

como se estivéssemos multiplicando o númerador e o denominador pelo conjugado do denominador:

\begin{equation*} \frac{z}{w} = \frac{a+bi}{c+di} = \frac{(a+bi)(c-di)}{(c+di)(c-di)} = \frac{(ac+bd)+(bc-ad)i}{(c^2+d^2)+(cd-cd)i}= \frac{ac+bd}{c^2+d^2} + \frac{bc-ad}{c^2+d^2} i. \end{equation*}

Representamos o número complexo \(z=a+bi\) como o vetor \((a,b)\) em \(\mathbb{R}^2\text{,}\) que será chamado de plano complexo nesse contexto. Essa identificação realiza o módulo (ou norma), \(|z|\text{,}\) de \(z=a+bi\) como o comprimento do vetor \((a,b)\) e o valor principal do argumento, \(\theta\text{,}\) como o ângulo \(\theta \in [0,2\pi)\) que \((a,b)\) faz com semi-eixo positivo:

\begin{align*} \amp |z|=\sqrt{a^2+b^2}=\sqrt{z\bar{z}},\\ \amp \mbox{se } a \neq 0, \,\,\, \theta = \mbox{arctan} \left(\frac{b}{a}\right), \\ \amp \mbox{se } a = 0 \mbox{ e } \, b > 0, \,\,\, \theta= \frac{pi}{2}, \\ \amp \mbox{se } a = 0 \mbox{ e } \, b < 0, \,\,\, \theta = -\frac{pi}{2}, \end{align*}

onde o argumento é dado em radianos.

Nota 21.2.
Se o valor principal do argumento de \(z\) é \(\theta\text{,}\) temos que \(z =|z|(\mbox{cos}(\theta) + i\mbox{sen}(\theta))\text{.}\) Para qualquer \(n \in \mathbb{Z}\text{,}\) o vetor \(|z|(\mbox{cos}(\theta+n2\pi) + i\mbox{sen}(\theta+n2\pi))=z\text{,}\) ou seja, o mesmo número complexo pode ser escrito de infinitas formas. Para qualquer \(n \in \mathbb{N}\text{,}\) \(\theta+n2\pi\) é um argumento do número complexo \(z\text{.}\) A representação \(z =|z|(\mbox{cos}(\theta) + i\mbox{sen}(\theta))\text{,}\) para um argumento \(\theta\) qualquer, é chamada de forma polar do número complexo.

Recordando a representação da série de Taylor centrada em 0 da função exponencial \(e^x = \sum x^n/n!\text{,}\) podemos aplicar essa fórmula em um número imaginário \(ix\text{,}\) para obter

\begin{align*} e^{ix} = \amp \sum_{n=0}^\infty \frac{(ix)^n}{n!} = 1 +ix -\frac{x^2}{2!} -i\frac{x^3}{3!} + \cdots \\ = \amp 1 -\frac{x^2}{2!} + \cdots + ix -i\frac{x^3}{3!} + \cdots\\ = \amp \sum_{n=0}^ \infty \frac{(-1)^nx^{2n}}{(2n)!} + i\sum_{n=0}^ \infty \frac{(-1)^nx^{2n+1}}{(2n+1)!} \\ = \amp \mbox{cos}(x)+i\mbox{sen}(x). \end{align*}
Nota 21.3. Fórmula de Euler.
\begin{equation*} e^{ix} = \mbox{cos}(x)+i\mbox{sen}(x). \end{equation*}

As propriedades do produto e do quociente de duas exponenciais são estendidas para números imaginários:

\begin{align*} e^{i\theta}e^{i\phi} = \amp ( \mbox{cos}(\theta)+i\mbox{sen}(\theta))( \mbox{cos}(\phi)+i\mbox{sen}(\phi)) \\ = \amp ( \mbox{cos}(\theta)\mbox{cos}(\phi) -\mbox{sen}(\theta)\mbox{sen}(\phi))+i( \mbox{sen}(\theta)\mbox{cos}(\phi)+\mbox{cos}(\theta)\mbox{sen}(\phi))\\ = \amp \mbox{cos}(\theta+\phi)+i\mbox{sen}(\theta+\phi),\\ \frac{e^{i\theta}}{e^{i\phi}} = \amp \frac{e^{i\theta}e^{i(-\phi)}}{e^{i\phi}e^{i(-\phi)}} = \frac{( \mbox{cos}(\theta)+i\mbox{sen}(\theta))( \mbox{cos}(\phi)-i\mbox{sen}(\phi))}{ ( \mbox{cos}(\phi)+i\mbox{sen}(\phi))( \mbox{cos}(\phi)-i\mbox{sen}(\phi))} \\ = \amp \frac{( \mbox{cos}(\theta)\mbox{cos}(\phi) +\mbox{sen}(\theta)\mbox{sen}(\phi))+i( \mbox{sen}(\theta)\mbox{cos}(\phi)-\mbox{cos}(\theta)\mbox{sen}(\phi))}{\mbox{cos}^2(\phi)+\mbox{sin}^2(\phi)}\\ = \amp \mbox{cos}(\theta-\phi)+i\mbox{sen}(\theta-\phi) \end{align*}
Nota 21.4.
A Fórmula de Euler nos dá uma maneira prática de representar números complexos na forma polar e nos permite ver, que ao multiplicarmos dois números complexos, estamos multiplicando os módulos dos dois e somando os ângulos deles. Se \(z = |z|e^{i \theta}\) e \(w = |w|e^{i \phi}\text{,}\) então
\begin{equation*} zw=|z|e^{i \theta} |w|e^{i \phi} = |z||w|e^{i \theta} e^{i \phi}= |z||w|e^{i (\theta + \phi)} \end{equation*}
e
\begin{equation*} \frac{z}{w}=\frac{|z|}{|w|}\frac{e^{i \theta}}{e^{i \phi}}=\frac{|z|}{|w|}e^{i (\theta- \phi)}. \end{equation*}

Desse modo, para \(z = |z|e^{i \theta}\) , obtemos a Fórmula de Moivre

\begin{equation*} z^n = |z|^n e^{i n\theta} = |z|^n (\mbox{cos}(n\theta)+i\mbox{sen}(n\theta)), \end{equation*}

o que nos permite verificar 2  que as raízes \(n\)-ésimas de \(z = |z|e^{i \theta}\) são

\begin{align*} \amp z_k = |z|^{1/n}e^{i(\theta + k2\pi)/n} = |z|^{1/n} (\mbox{cos}((\theta + k2\pi)/n)+i\mbox{sen}((\theta + k2\pi)/n),\\ \amp k \in \{0, \ldots, n-1\} \end{align*}
