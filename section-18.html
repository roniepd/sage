Seção 18 Testes da Razão e da Raiz

Subseção 18.1 O Teste da Razão

O teste da razão está relacionado à determinação do raio de convergência de uma série de potências (que veremos mais adiante) e será importante nas aplicações de séries de potências.

Note que para uma progressão geométrica, \(a_n = a r^n\text{,}\) a razão é o resultado do quociente de um termo pelo próximo

\begin{equation*} \frac{a_{n+1}}{a_n} = r. \end{equation*}

Por outro lado recordamos que a série geométrica é convergente se \(-1 < r < 1\) e divergente, caso contrário. A situação não é tão simples para uma série qualquer, mas o quociente \(\frac{a_{n+1}}{a_n}\) ainda contém informações sobre a convergência da série.

Teorema 18.1. Teste da Razão.

Seja \(\sum a_n\) uma série:

(i) Se \(\lim\limits_{n \to \infty} \left|\frac{a_{n+1}}{a_n}\right| = L < 1\text{,}\) então a série \(\sum_{n=1}^\infty a_n\) é absolutamente convergente.

(ii) Se \(\lim\limits_{n \to \infty} \left|\frac{a_{n+1}}{a_n}\right| = L > 1\text{,}\) então a série \(\sum_{n=1}^\infty a_n\) é divergente.

(iii) Se \(\lim\limits_{n \to \infty} \left|\frac{a_{n+1}}{a_n}\right| =1\text{,}\) o teste é inconclusivo.

Demonstração.

(i) Consideremos um número \(r\text{,}\) \(L < r < 1\text{.}\) Como \(\lim\limits_{n \to \infty} \left|\frac{a_{n+1}}{a_n}\right| = L < r\text{,}\) existe \(N \in \mathbb{N}\text{,}\) tal que

\begin{equation*} n > N \,\,\, \Rightarrow \,\,\, \left|\frac{a_{n+1}}{a_n}\right| < r. \end{equation*}

Assim

\begin{equation*} n > N \,\,\, \Rightarrow \,\,\, |a_{n+1}|< r |a_n| \end{equation*}

e

\begin{align*} |a_{N+1}| \amp < r |a_N|,\\ |a_{N+2}| \amp < r |a_{N+1}| < r^2 |a_N|,\\ \vdots\amp\\ |a_{N+k}| \amp < r^k |a_N|,\,\,\,\, \forall k \geq 1. \end{align*}

Por outro lado, como \(0 < r < 1,\) \(\sum_{k=1}^\infty |a_{N}|r^k\) é convergente. Então, pelo teste da comparação 1 , a série \(\sum_{k=1}^\infty |a_{N+k}| \) é convergente. Então a série \(\sum a_n\) é absolutamente convergente.

com \(b_n = r^k |a_N|\)

(ii) Consideremos um número \(r\text{,}\) \(1 < r < L\text{.}\) Como \(\lim\limits_{n \to \infty} \left|\frac{a_{n+1}}{a_n}\right| = L > r\text{,}\) existe \(N \in \mathbb{N}\text{,}\) tal que

\begin{equation*} n > N \,\,\, \Rightarrow \,\,\, \left|\frac{a_{n+1}}{a_n}\right| > r. \end{equation*}

Assim

\begin{equation*} n > N \,\,\, \Rightarrow \,\,\, |a_{n+1}|> r |a_n| \end{equation*}

e

\begin{align*} |a_{N+1}| \amp > r |a_N|,\\ |a_{N+2}| \amp > r |a_{N+1}| < r^2 |a_N|,\\ \vdots\amp\\ |a_{N+k}| \amp > r^k |a_N|,\,\,\,\, \forall k \geq 1. \end{align*}

Como \(r^k |a_N| \to \infty\text{,}\) \(|a_{N+k}| \) não tende a 0 e \(\sum a_n\) é divergente.

Exercício 18.2.

Verifique se as séries seguintes são convergentes:

  1. \(\displaystyle \sum_{n=1}^\infty \frac{n^2}{2^n};\)

  2. \(\displaystyle \sum_{n=1}^\infty \frac{1}{n!}.\)

O código pode ser utilizado para estimar o quociente \(a_{n+1}/a_n\text{.}\)

Exercício 18.3.

Determine para que valores \(x \in \mathbb{R}\) as séries seguintes são convergentes:

  1. \(\displaystyle \sum_{n=1}^\infty \frac{x^n n^2}{2^n};\)

  2. \(\displaystyle \sum_{n=1}^\infty \frac{x^n}{n!}.\)

Subseção 18.2 O Teste da Raiz

Note que para uma progressão geométrica, \(a_n = a r^n\text{,}\) o módulo da razão é o resultado do seguinte limite

\begin{equation*} \lim\limits_{n \to \infty} \sqrt[n]{|a_n| } = |r|. \end{equation*}

Por outro lado recordamos que a série geométrica é convergente se \(-1 < r < 1\) e divergente, caso contrário. Para uma série qualquer, o valor \(\lim\limits_{n \to \infty} \sqrt[n]{|a_n| } \) contém informações sobre a convergência da série.

Teorema 18.4. Teste da Raiz.

Seja \(\sum a_n\) uma série:

(i) Se \(\lim\limits_{n \to \infty} \sqrt[n]{|a_n| }= L < 1\text{,}\) então a série \(\sum_{n=1}^\infty a_n\) é absolutamente convergente.

(ii) Se \(\lim\limits_{n \to \infty} \sqrt[n]{|a_n| }= L > 1\text{,}\) então a série \(\sum_{n=1}^\infty a_n\) é divergente.

(iii) Se \(\lim\limits_{n \to \infty} \sqrt[n]{|a_n| }= 1\text{,}\) o teste é inconclusivo.

Demonstração.

(i) Consideremos um número \(r\text{,}\) \(L < r < 1\text{.}\) Como \(\lim\limits_{n \to \infty} \sqrt[n]{|a_n| } = L < r\text{,}\) existe \(N \in \mathbb{N}\text{,}\) tal que

\begin{equation*} n > N \,\,\, \Rightarrow \,\,\, \sqrt[n]{|a_n| }< r. \end{equation*}

Assim

\begin{equation*} n > N \,\,\, \Rightarrow \,\,\, |a_{n}|< r^n. \end{equation*}

Por outro lado, como \(0 < r < 1,\) \(\sum_{n=1}^\infty r^n\) é convergente. Então, pelo teste da comparação 2 , a série \(\sum_{n=1}^\infty |a_{n}| \) é convergente. Então a série \(\sum a_n\) é absolutamente convergente.

com \(b_n = r^n\)

(ii) Consideremos um número \(r\text{,}\) \(1 < r < L\text{.}\) Como \(\lim\limits_{n \to \infty} \sqrt[n]{|a_n| }= L > r\text{,}\) existe \(N \in \mathbb{N}\text{,}\) tal que

\begin{equation*} n > N \,\,\, \Rightarrow \,\,\,\sqrt[n]{|a_n| } > r. \end{equation*}

Assim

\begin{equation*} n > N \,\,\, \Rightarrow \,\,\, |a_{n}|> r^n. \end{equation*}

Como \(r^n \to \infty\text{,}\) \(|a_{n}| \) não tende a 0 e \(\sum a_n\) é divergente.

Exercício 18.5.

Determine para que valores \(x \in \mathbb{R}\) as séries seguintes são convergentes:

  1. \(\displaystyle \sum_{n=1}^\infty \frac{x^n(-1)^n}{2^n};\)

  2. \(\displaystyle \sum_{n=1}^\infty \frac{3^n x^n}{5}.\)
